1 User attempts to create an outgoing FCY remittance transaction in T24

T24 checks the receiver country/entity against the caution notice list:

  • If receiver country has Caution Notice A, allow transaction but display warning
  • If receiver bank has Caution Notice B, allow transaction but display warning
  • If receiver country has Caution Notice C, display warning and require override
  • If receiver entity has Caution Notice B and receiver country has Caution Notice C, display country warning and require override

Transactions are processed according to the appropriate caution notice level, with warnings displayed and overrides required as per MUFG requirements.

Transactions involving high-risk countries/entities under Caution Notice C undergo additional scrutiny before being allowed.
2 User attempts to amend an existing outgoing FCY remittance transaction in T24 repair screen

T24 re-evaluates the caution notice status based on the amended transaction details:

  • If new receiver country has Caution Notice A, allow amendment but display warning
  • If new receiver bank has Caution Notice B, allow amendment but display warning
  • If new receiver country has Caution Notice C, display warning and require override
  • If new receiver entity has Caution Notice B and new receiver country has Caution Notice C, display country warning and require override

Amended transactions are checked against caution notices, with appropriate warnings and overrides required based on the new transaction details.

High-risk amendments involving Caution Notice C countries/entities undergo scrutiny before being allowed.
